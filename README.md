# crystal-omp

OpenMP integration for Crystal.  Supported up to latest version.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     crystal-omp:
       github: Oblivious-Oblivious/crystal-omp
   ```

2. Run `shards install`
